# Examples — JQB_WindowsLib

This directory contains complete example applications demonstrating the library's capabilities.

| Example | Description | Components |
|---------|-------------|------------|
| [01_hello_window](01_hello_window.md) | Minimal window with label and button | SimpleWindow, Label, Button |
| [02_serial_monitor](02_serial_monitor.md) | COM port monitor with logging | SimpleWindow, Select, Button, TextArea, Serial |
| [03_form_inputs](03_form_inputs.md) | Form with inputs, checkboxes, select | InputField, CheckBox, Select, Button |
| [04_realtime_dashboard](04_realtime_dashboard.md) | Dashboard with chart and display | Chart, ValueDisplay, ProgressBar |
| [05_tabbed_app](05_tabbed_app.md) | Application with tabs | TabControl, Label, Button |
| [06_ble_scanner](06_ble_scanner.md) | Bluetooth LE device scanner | BLE, Select, TextArea, Button |
| [07_image_viewer](07_image_viewer.md) | Image viewer | ImageView, Button, Select |
